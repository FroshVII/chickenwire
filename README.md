A library for representing and manipulating hexagonal grids in Rust.

## Installation
Add the following to your project's `Cargo.toml` file under `[dependencies]`:

```toml
chickenwire = "0.1.0"
```

Check out [Chickenwire's Cargo page]() for the latest version info.
